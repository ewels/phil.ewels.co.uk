Projects

  • nf-core
    • A community effort to collect a curated set of analysis pipelines built using Nextflow.
  • MultiQC
    • Aggregate results from bioinformatics analyses across many samples into a single report.
  • SRA Explorer
    • Mini web application to explore the NCBI SRA. Generates SRA/FastQ download URLs.
  • Cluster Flow
    • A simple and flexible bioinformatics pipeline tool.
  • QC Fail
    • Articles written about common next-generation sequencing problems.
  • 10XQC
    • A web tool to crowdsource QC metrics from 10X Cell Range reports.
  • Labrador
    • A web interface to manage and automate the processing of publicly available datasets.
  • WordPress Plugins
    • A handful of plugins that I've written and released for general use.
